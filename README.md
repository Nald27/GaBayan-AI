# GaBayan AI

GaBayan AI is a modern legal guidance assistant designed to help users in the Philippines prepare better questions before consulting a public attorney or licensed lawyer.

## Features

- AI-powered legal guidance assistant
- Helps users organize legal concerns and documents
- Provides general legal information
- Guides users on what to ask a public attorney
- Modern dark minimalist UI
- Mobile responsive
- Built with React, Vite, Express, and Gemini AI

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js
- Google Gemini API

## Important Disclaimer

GaBayan AI provides general legal information only and is not a substitute for advice from a licensed lawyer or public attorney.


